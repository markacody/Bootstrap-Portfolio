About Me

What I Do

Generalist

As a career educator, two questions summarize almost all that I do.

"How can this be easier?"
"How can this be fun?"

With these questions in mind, I am an analyst, designer, and developer of solutions for learning, performance, and enhanced user experience.

Analyst

Analysis is about understanding the user experience. When it comes to the web and to information systems, a temptation is to limit analysis to a system and its user interface, but experience is shaped by events that occur before, during, and after a transaction.

Designer

Stories and expectations matter. Impact is everything. A solution that takes hold and makes change is more than an app. It's a coordinated system of communications, events, and interactions. Design can lead to a better version of real life. And more compelling.

Developer

In a small learning design shop, access to professional IT resources is limited, so I provide the web programming and back end skills. I am a User Experience Designer and Full Stack Web Developer.
